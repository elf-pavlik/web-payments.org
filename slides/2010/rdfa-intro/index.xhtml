The Problem

Computers have a hard time understanding human concepts in Web pages.


Why: Increase search engine accuracy/ranking, universal data interoperability, better interfaces for the Web.

Describing everything...

The Resource Description Framework (RDF)

  • Purpose: A model to describe almost every human concept to computers.
  • Based on a simple concept - statement of fact: Subject Property Object.
  • For example: article title "Man Walks on the Moon" .
  • As RDF: <http://example.org/article-1> <http://purl.org/dc/terms/title> "Man Walks on the Moon" .

Serializing RDF...

The Resource Description Framework in attributes (RDFa)

  • Purpose: A way to express human concepts in HTML so that machines can understand them.
  • RDFa uses HTML attributes like about and property to express RDF.
  • HTML+RDFa example:
    <p about="http://example.org/article-1">
       <span property="http://purl.org/dc/terms/title">Man Walks on the Moon<span>
    <p>
  • Result: <http://example.org/article-1> <http://purl.org/dc/terms/title> "Man Walks on the Moon" .
