Web

Commerce

PaySwarm

PaySwarm the first open, universal standard for payments on the Web.

  • Payment provider agnostic
  • Currency agnostic - USD, Euro, Bitcoins, etc.
  • Zero-click payments, digital contracts and receipts
  • Decentralized and extensible
  • Commercialized: Meritora

Web

Commerce

Web Commerce API

The Web Commerce API is a way to initiate transactions from the browser.

  • Make payment native and easy - one click
  • Protect from phishing attacks
  • Dumb pipe - supports both proprietary payment schemes as well as PaySwarm
  • Commercialized: FirefoxOS
