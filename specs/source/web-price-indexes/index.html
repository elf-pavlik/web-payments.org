Price Index Web Services

A Basic Price Index Query

A basic price index query contains at least ...

@context
The value https://w3id.org/payswarm/v1, or an equivalent JSON-LD Context expressing the terminology necessary for the expression of the data.
@type
The value PriceIndexResult.

A Price Index Response

A typical price index response will look like... For example: 

{
  "@context": "https://w3id.org/payswarm/v1",
  "@type": "PriceIndexResult",
  "amount": "23.41",
  "currency": "USD"
}
